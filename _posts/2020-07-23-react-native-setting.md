# React Native

## Requiremnets

### 1. Node

노드 설치 : https://nodejs.org/ko/

설치확인
```
node -v
```

### 2. NPM

설치 확인
```
npm -v
```

### 3. NPX

설치 확인
```
npx -v
```

### 4. Expo

```
npn install -g expo-cli
```

### 5. Expo Client App

App Store나 Google Play Store에서 다운로드
