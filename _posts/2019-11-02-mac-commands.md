### Mac 단축키

#### 기본
  
| 단축키 | 동작 |
|:----|:----|
| Command + c | 복사 |
| Command + x | 잘라내기 |
| Command + v | 붙여넣기 |
| Command + r | 새로고침 |
| Command + ~ | 앱 내 창전환 |
| Control + Command + Space | 특수문자 |
| Control + Tab | 우측탭 이동 |
| Control + Shift + Tab | 좌측탭 이동 |
| fn + f11 | 데스크탑 보기 |
| Control + ↑ | Mission Control |
| Control + → | 우측 Space 이동 |
| Control + ← | 좌측 Space 이동 |
| Command + R | Refresh |
| Command + Shift + 3 | 캡처 |
| Command + Shift + 4 | 영역 캡처 |

#### Finder
  
| 단축키 | 동작 |
|:----|:----|
| Command + ↓ or O | 실행 |
| Command + ↑ | 상위폴더 |
