## Mac 초보의 Mac Pro 적응기

### Mac 단축키

#### 기본
  
| 단축키 | 동작 |
|:----|:----|
| Command + c | 복사 |
| Command + x | 잘라내기 |
| Command + v | 붙여넣기 |
| Command + r | 새로고침 |
| Command + ~ | 앱 내 창전환 |
| Control + Command + Space | 특수문자 |

#### Finder
  
| 단축키 | 동작 |
|:----|:----|
| Command + ↓ or O | 실행 |
| Command + ↑ | 상위폴더 |
