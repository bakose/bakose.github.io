갤럭시 S10에서의 개발자 옵션 활성화 하는 방법입니다.

1. 설정 > 휴대전화 정보 선택

2. 소프트웨어 정보 

3. 빌드번호 항목을 5번 이상 연속으로 탭

4. 설정 화면의 최하단에 개발자 옵션 메뉴 활성화 됨
